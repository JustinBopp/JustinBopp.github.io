Common Questions

  • Do I need to contact advisors before applying? It depends on the field and program, but reaching out can be useful if you have a real research-fit question and have read enough to ask something specific.
  • How many schools should I apply to? There is no single right number, but it helps to have a balanced list with a realistic spread of programs, advisors, and costs.
  • When should I ask for letters? Earlier than you think. Give letter writers time, context, and a clear deadline list.
  • Should I apply right away if I feel underprepared? Sometimes yes, but sometimes a bridge program or post-bacc will set you up much better for the next cycle.

Where to Go Next

If you are trying to figure out timing or organization, start with the Planning Guide. If you need outside links, go to Application Resources. If you are deciding whether to wait and strengthen your application, go to Alternatives.

Scope

These answers are mostly tuned to physics and astronomy applications, so some norms can look different in other fields.
