# JustinBopp.github.io

This repo now uses a simple multi-page static site layout for GitHub Pages.

Main pages:

- `/index.html`
- `/about/index.html`
- `/research/index.html`
- `/cv/index.html`

Research subpages:

- create folders inside `/research/`
- each project gets its own `index.html`

Hidden pages:

- create folders inside `/hidden/`
- do not link them from the visible site if you want direct-link-only access

Media:

- upload videos to `/assets/media/`
- upload PDFs and downloadable files to `/assets/files/`
