Editing Help

What changed from the old template?

The old site relied on popup sections inside a single HTML file. This new setup uses one file per page, which makes research projects, subpages, and media management much easier.

  • Add a new research project by copying an existing folder inside research/.
  • Add plots by uploading images and duplicating a gallery block on a project page.
  • Add videos by uploading an .mp4 file and updating the sample video block.
  • Create direct-link-only pages inside hidden/ and do not add them to navigation.
